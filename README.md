# 💡 Light Out

A simple yet addictive puzzle game where the goal is to turn off all the lights on the board.

Every tap toggles the selected light and its adjacent lights, making each move important. Plan carefully and solve the puzzle in as few moves as possible.

## 🎮 Gameplay

The objective is simple:

* Tap a light to toggle it on or off.
* Adjacent lights are also affected by each move.
* Turn off all lights to win the game.
* Challenge yourself to solve puzzles using the fewest moves possible.

## 🚀 Getting Started

### Prerequisites

* Node.js
* npm or yarn
* Expo CLI (if using Expo)

### Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/light-out.git
```

Navigate to the project directory:

```bash
cd light-out
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npx expo start
```

Run on Android:

```bash
npx expo run:android
```

Run on iOS:

```bash
npx expo run:ios
```
